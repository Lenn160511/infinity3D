Über infinity3D

Ein Kinderzimmer,
ein Drucker, eine Idee.

infinity3D wurde von einem 15-jährigen Tüftler gegründet, der irgendwann einfach wissen wollte, wie aus einer Datei ein echtes, anfassbares Objekt wird. Aus der Neugier ist eine kleine Werkstatt geworden, in der mittlerweile für Freunde, Familie und alle, die eine Idee haben, gedruckt wird.

Jeder Auftrag wird von Hand vorbereitet, ausgerichtet und überwacht – kein automatisierter Großbetrieb, sondern echtes Interesse an jedem einzelnen Druck.
